Steps on how to scetch and draw

  • Before anything else lets show the end result of the steps to see what we are aiming for(hover to enlarge).

  • Step 1

    Find a reference to what you are aiming to draw, it makes drawing certain object alot easier

  • Step 2

    Use an image painting program/manipulator like Paint tool Sai,MS Paint,Photoshop etc.

    I use the program called Paint tool Sai, make a new canvas and create a lineart layer

  • Step 3.

    Using the curve tool start drawing your object until you get the whole lineart of the drawing,that you can actualy color when you are done.

    After you are done with the main scetch you can add other object that are simpler.And start coloring by painting the scetch in main simple colors on layer 1 not on the line art layer

    Then just add lights and shadows depending on where you want your light to come from.

  • Step 4.

    Get a background or draw your self one, and add it as a new layer to the canvas by pasting or drawing it.When your ready merge the lineart layer with the color layer.

    Then make the background layer to be on multiply mode.

    And you get the final result.
